# Net Package

The `Net` package groups networking domains under a single entry point. These modules cover outbound HTTP/HTTPS, inbound servers, raw sockets, and stream utilities.

```aivi
use aivi.net
```

## Modules

- `aivi.net.http`
- `aivi.net.https`
- `aivi.net.http_server`
- `aivi.net.sockets`
- `aivi.net.streams`
